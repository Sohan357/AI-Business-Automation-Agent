# 🤖 AI Business Automation Agent

### Transforming CRM, HRMS & Project Management with AI

AI Business Automation Agent is an intelligent business operations platform that unifies **Customer Relationship Management (CRM)**, **Human Resource Management (HRMS)**, and **Project Management (PMS)** into a single AI-powered solution.

Built using **Python, Streamlit, LangChain, Ollama, and Llama 3**, the platform automates repetitive business tasks, provides actionable insights, and enhances decision-making through Artificial Intelligence.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sohan357/AI-Business-Automation-Agent.git
cd AI-Business-Automation-Agent
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
